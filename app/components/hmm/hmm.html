The Hidden Markov Model

Let's start off by defining a Hidden Markov Model. There isn't a whole lot to it apart from a few different types of probabilities. The first is the start probability, that is, the probability that the sequence starts in that state. Next up is transition probability. That is the general probability that a state (say Sunny) turns into any other state (say Raining). Last are the probabilities of being in any state given an observation.
